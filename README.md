# spring-boot-gwt
Example application using Spring-Boot 2.0.0, Jersey, & GWT 2.8.2.

## Quick Start

    mvn clean package
 
From first terminal:
 
    mvn spring-boot:run
    
From second terminal:

    mvn gwt:run
    
Open [http://localhost:8080/index.html](http://localhost:8080/index.html) in your browser.
